## Overview

drasi-lib is a Rust crate that brings Drasi's change detection capabilities directly into your Rust applications. This guide will walk you through adding drasi-lib to your project and running your first continuous query.

## Prerequisites

- Rust 1.70 or later
- Cargo package manager

## Installation

Add drasi-lib to your `Cargo.toml`:

```toml
[dependencies]
drasi-lib = "0.1"
```
