## What You'll Learn

By completing this path, you will:

- Understand Drasi deployment options and choose the right one
- Deploy Drasi to a production Kubernetes environment
- Set up comprehensive observability with metrics and tracing
- Configure monitoring, alerting, and dashboards
- Know how to scale Drasi for production workloads
- Be equipped to troubleshoot common issues

## Prerequisites

- Experience with Kubernetes and container orchestration
- Familiarity with monitoring and observability tools
- Access to a Kubernetes cluster for deployment
- About 3-4 hours of dedicated time
