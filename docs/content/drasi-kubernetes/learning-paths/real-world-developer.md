## What You'll Learn

By completing this path, you will:

- Build a complete real-time monitoring application
- Configure production-grade data sources
- Set up real-time reactions for UI updates
- Connect frontend applications to Drasi
- Apply query design best practices

## Prerequisites

- Completed the [Getting Started Path](../getting-started/) or equivalent experience
- Familiarity with web development concepts
- About 4-6 hours of dedicated time

## Next Steps

After completing this path, consider:

- [Operations Path](../operations/) - Learn to deploy and maintain Drasi in production
- [Architecture Path](../architecture/) - Design enterprise-scale solutions
