## What You'll Learn

By completing this path, you will:

- Understand the available source types and when to use each
- Know all reaction options for outputting query results
- Integrate Drasi with Dapr for microservices scenarios
- Set up event-driven reactions with Azure Event Grid
- Connect Drasi to downstream systems
- Build custom sources and reactions for your specific needs

## Prerequisites

- Completed the [Getting Started Path](../getting-started/) or equivalent experience
- Familiarity with the systems you want to integrate
- Understanding of event-driven architecture concepts
- About 2-3 hours of dedicated time
