## What You'll Learn

By completing this path, you will:

- Configure data sources for change detection
- Write effective continuous queries
- Build real-time dashboards with SSE
- Send change notifications via HTTP webhooks
- Apply query design best practices

## Prerequisites

- Completed the [Quick Start Path](../getting-started/) or equivalent experience
- Familiarity with web development concepts
- About 3-4 hours of dedicated time
