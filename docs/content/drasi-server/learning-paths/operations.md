## What You'll Learn

By completing this path, you will:

- Understand Drasi Server deployment options
- Deploy Drasi Server with Docker Compose
- Configure production settings
- Monitor and troubleshoot Drasi Server
- Use the CLI and REST API for management

## Prerequisites

- Experience with Docker and container orchestration
- Familiarity with configuration management
- About 2-3 hours of dedicated time
