## What You'll Learn

By completing this path, you will:

- Have a working Drasi Server installation
- Understand the core concepts: Sources, Continuous Queries, and Reactions
- Set up a local development environment
- Monitor a PostgreSQL database for changes
- Know the configuration options available

## Prerequisites

- Basic familiarity with databases and SQL concepts
- Docker installed on your machine
- About 1-2 hours of dedicated time
