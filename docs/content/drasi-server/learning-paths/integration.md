## What You'll Learn

By completing this path, you will:

- Understand the available source types for Drasi Server
- Configure HTTP sources to receive webhook events
- Set up gRPC sources for high-performance streaming
- Use Platform sources with Redis streams
- Push real-time updates to browsers with SSE
- Stream changes to clients via gRPC
- Bootstrap queries with existing data

## Prerequisites

- Completed the [Quick Start Path](../getting-started/) or equivalent experience
- Familiarity with the systems you want to integrate
- Understanding of event-driven architecture concepts
- About 2-3 hours of dedicated time
